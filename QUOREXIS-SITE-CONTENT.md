# 6) DIRECTION VISUELLE

## Palette de couleurs

| Nom           | Hex       | Usage                                           |
|---------------|-----------|--------------------------------------------------|
| Noir profond  | `#0A0A0A` | Texte principal, headlines                       |
| Blanc cassé   | `#F7F7F5` | Fond principal (background)                      |
| Gris moyen    | `#6B6B6B` | Texte secondaire, descriptions, sous-titres      |
| Accent sobre  | `#2C3E50` | Boutons, liens actifs, bordures hover            |

> Pas de couleur vive. Pas de gradient. Le contraste se fait par la typographie et l'espace.

## Typographies

| Usage            | Recommandation                  | Fallback         |
|------------------|---------------------------------|------------------|
| Headlines        | **Inter** (Medium / SemiBold)   | Helvetica Neue   |
| Body / paragraphes | **Inter** (Regular / Light)   | System sans-serif |

> Une seule famille suffit. Inter est disponible nativement dans Framer.
> Alternative si on veut un accent éditorial : **DM Sans** pour les headlines.

## Règles de mise en page

| Paramètre                | Valeur recommandée                        |
|--------------------------|-------------------------------------------|
| Largeur max du contenu   | 1200px                                    |
| Padding horizontal       | 80px (desktop), 24px (mobile)             |
| Espacement entre sections| 120px–160px (desktop), 80px (mobile)      |
| Grille expertises        | 3 colonnes (desktop), 1 colonne (mobile)  |
| Cards                    | Bordure 1px `#E0E0E0`, radius 8px, padding 32px, pas d'ombre ou ombre très légère (`0 1px 3px rgba(0,0,0,0.04)`) |
| Taille headline hero     | 48–56px (desktop), 32px (mobile)          |
| Taille body              | 17–18px, line-height 1.6                  |
| Taille sous-titres       | 20px, line-height 1.5                     |

## Visuels Hero — 3 idées

### Idée 1 : Lignes architecturales
Un réseau de lignes fines parallèles et perpendiculaires sur fond clair, évoquant une structure technique. Tons gris et blanc. Impression de précision et d'ordre.

### Idée 2 : Topographie abstraite
Des courbes de niveau très fines, comme une carte topographique minimaliste. Monochrome (gris sur blanc cassé). Évoque la profondeur, la méthode, la cartographie d'un système.

### Idée 3 : Grille en perspective
Une grille isométrique subtile qui s'estompe vers les bords. Noir très léger sur fond blanc. Évoque l'infrastructure, la scalabilité, la rigueur.
